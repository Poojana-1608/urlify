# URLify - URL Shortener

## Project Description
URLify is a URL shortening application that allows users to create short URLs, manage them, and track clicks.

## Features
- User Registration
- User Login
- Create Short URLs
- View URL History
- Delete URLs
- Click Analytics

## Technologies Used
- React.js
- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication
